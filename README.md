# OceanRithm Documentation

This repository contains the official documentation website for  
**OceanRithm Solutions UG (haftungsbeschränkt)**.

- Built with: MkDocs + Material theme  
- Live website: https://oceanrithm.com  
- Language support: EN / DE  
- Legal pages included (Impressum, Privacy Policy, Disclaimer)

## Development

Install MkDocs:

pip install mkdocs mkdocs-material

Run locally:

mkdocs serve

Deploy to GitHub Pages:

mkdocs gh-deploy
