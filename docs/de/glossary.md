# Glossar

### Governance  
Übergeordnete konzeptionelle Regulierung des Workflow Verhaltens.

### Tenant  
Eine konzeptionelle Darstellung eines Benutzers oder einer logischen Gruppe.

### Abstraktion  
Fokussierung auf Ideen statt auf Implementierungsdetails.

### Workflow  
Ein nicht ausführbarer, konzeptioneller Ablaufpfad.

### Beobachtung  
Ein übergeordnetes Monitoring Konzept für Forschungszwecke.
